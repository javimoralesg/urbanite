void _timer_pwm_buzzer_config ( uint32_t  buzzer_id)

Configure the timer that controls the PWM of the buzzer system.

This function is called by the port_buzzer_init() public function to configure the timer that controls the PWM of the buzzer.

Note
The duty cycle of the PWM signal is not set yet. This is done in the port_buzzer_set_sound() function where the sound is set.
Parameters
buzzer_idBuzzer system identifier number.
